# Eye widget
 ### Виджет для конкурса Волга-IT`22
 
***Eye widget*** - это виджет, предназначенный для индивидуального подбора очков через заполнение интерактивной формы.

# Особенности виджета:

 * Подключение в любой сайт
 * Адаптивность виджета под все экраны
 * Кроссбраузерность
